BASIC PAGE

This page is for the basic principles of NodeJS, such as arrays, JSON and variables
    The point is to give a simple and clear explaniation of the subjects with understandable examples.

                  

                    
Variables
Variables in NodeJS has the same functions as in most other languages. It is an container that hold values.
          We use 3 different kind of variables. Var, let and const. We will distinguish between mutable and imutalbe variables.
          A mutable variable is one, that can be modified after it is created, where an imutable can not. 
          
          > var & let: Both are mutable and is much alike with some differences. The main difference is the scope of the variables. 
          Var has it scope to functions, while let has block scope ie. to enclosing brackets ("{}"). Example: 
                    
                      function run() {
                        var foo = "Foo";
                        let bar = "Bar";
                      
                        console.log(foo, bar); // Foo Bar
                        {
                          var moo = "Mooo"
                          let baz = "Bazz";
                          console.log(moo, baz); // Mooo Bazz
                        }
                        console.log(moo); // Mooo
                        console.log(baz); // ReferenceError
                      }
                      run();
                    

          Here is the var scoped to the entire function, where let is only usable inside the enclosing brackets. 
          So the var in the middle ("Mooo") is used outside of the enclosing brackets as it log it in the end, with no problems
          as it is still inside of the function run(), but it throws an error for the let, as the console log
          for the let ("Bazz") is outside of the scope of the enclosing brackets ("{}").
          
          > Const: Is an imutable variable. Const works a lot like let which we have just talked about, with the 
          exception that it cant be changed after it has been created. That it works like let means that it also has block scope.
          Same example as above can be used with let being swapped with const.

              

                
Scope

          Scope is the extend of which a variable is visible to the program. All variable have a scope but not all variables
          have the same type of scope. we primarily work with 2 different types of scope. Function- and blocks scope.
          > function scope:  Where the variable is visible inside of an entire function.
          > block scope: Where the variable is visible inside of the enclosing brackets it is inside.          
             

            
Hoisting

          Hoisting is an important thing to be aware of when you talk about scope. Hoisting is the name of an feature in javascript That
          that declares every variable inside a given scope in the beginning of the function. It does this even if the variable first 
          is declared later down the line of the function. Example: 
          
                    function f() {
                      console.log(bar);  // undefined
                      var bar = 'abc';
                      console.log(bar);  // abc
                    }
          

          So the bar variable is declared already before the first console.log expression but returns undefined as there is no value to bar yet.           
        

                
Object

          An object in NodeJS is a datatype and is one of the more complex data types in NodeJS. 
          Object is a class that almost all objects inherits from. 
          An Object is made up of key&Value pairs, that defines what the object is and what it does.
          Example of a new object of a "car": 
          
                let car = {type: "volvo", Horsepower: 2000, color:"Yellow"} 
          

          We have now created a new object called "car" with 3 key&value pairs that descripe the object. 
          We can then interact with our object by making calls on the objects keys to acces its values.
          
 
                car.type = "fiat";
                car.color = "black";
          

          We changed the values of the car by calling the keys, that belonged to that value. 

          we can also get the value of an object by calling the objects keys. 
          
 
                const newType = car.type();
          

              

                  
Arrays

            An Array is a data container that is used to storage values in a systematic order.
            We take values and input them into a list that is ordered by index numbers. 
            So the first index (always called 0) is the first and the last index is array.length. 
            We can store all values in an array, from simple values like 1, 5.2 and "HELLO" to more 
            complex values like objects. To interact with the elements of an array, will we have to acces them 
            via the index. Example:
                
                  // Creating array with 3 elements
                  let array = [1, 54, 1001]

                  // Getting the values of index 2
                  const value = array[1];

                  // Getting a specifik value in the array (returns 1 as that is the index where the value is)
                  array.indexOf(54)
                

            There is a lot of different ways to work with arrays. We can also iterate through an array with maps.
            Arrays can also be used as a simple database and a lot more.   
                

                    
Map

                    The map() creates a new array filled with the result of calling a function on every element of the calling array. 
                    Example: 
                    
                      const array = [1,2,3,4]; 

                      const newMapArray = array.map(x => x + 1); 
                      // Takes the array and for each element it add 1 and put it into the new array

                      console.log(newMapArray); // Output = [2,3,4,5]
                    

                    The important thing to remember, is that it takes data from one array, does something
                    and creates a new array populated with the manipulated data. 
                  

                      
Slice()

                      Slice() is a function that can copy specifik parts of an array to a new array.
                      We do that by using a start point and endpoint of elements that should be copied. 
                      Example: 
                      
                        const array = [1,2,3,4];

                        const newArray = array.splice(0,3); 
                        // Start from index 0 and goes to index 2 as the end point is exluding.

                        console.log(newArray); // output = []1,2,3]
                      

                      
Splice()

                      Splice is used to insert values into an array on specifik index.
                      Example: 
                      
                        const array = ["tomato", "cucumber", "rocket"];

                        array.splice(2, 0, "hamburger", "soda", "pizza"); 
                        // insert the values from index 2 of the array. 
                        // The 0 means that we dont want to delete anything before we insert values

                        console.log(array); // ["tomato", "cucumber", "hamburger", "soda", "pizza", "rocket"]
